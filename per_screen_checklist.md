# Screen Completion Checklist (per-screen-checklist.md)

- [ ] Matches wireframe specification exactly.
- [ ] Uses only design system tokens (no hardcoded hex/values).
- [ ] All loading/skeleton states implemented.
- [ ] Empty states and error handlers present.
- [ ] Fully responsive (1024px desktop, 390px mobile).
- [ ] Arabic RTL version validated (if applicable).
- [ ] Interactive states (hover, focus, active) complete.
- [ ] Connected to real-time event system.
- [ ] TGA compliance styling applied correctly.
- [ ] AI feature mocks/integrations wired.
- [ ] TypeScript types defined and exported.
- [ ] Basic unit/layout test passing.
