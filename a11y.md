# JAS Accessibility & QA Guide (a11y.md)

## Accessibility Standards
Target: **WCAG 2.1 AA Compliance**.

### 1. Color & Contrast
- Minimum contrast ratio of 4.5:1 for all UI text against the dark `#0F0F1A` base.
- Use of semantic color is always accompanied by an icon or clear text label (e.g., Green Checkmark for "Valid").

### 2. Keyboard Navigation
- All interactive elements must have a visible focus ring (`#7C5DDB` glow).
- Logical tab order through multi-step wizards and complex data tables.

### 3. Screen Reader Support
- ARIA labels for all icon-only buttons (e.g., "Dismiss Notification").
- Localized Arabic screen reader testing for the Driver App (VoiceOver/NVDA).

### 4. RTL Integrity
- Automated Playwright testing for every screen to ensure layout directionality and logical property compliance.
